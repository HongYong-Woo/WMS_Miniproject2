B2B 창고 관리 시스템(WMS) 개발

TOP(This is Our Page): 5명

## 개발 환경
`Java 17`
- **IDE**: IntelliJ IDEA Ultimate
- **Framework**: Spring Boot (3.x)
- **Database**: MySQL
- **Library**: Spring Security 6, MyBatis, Thymeleaf, Bootstrap
- **Tools**: GitHub, Notion, Slack

# 주요 기능
- 로그인/회원 관리
- 창고 관리
- 입고 관리
- 출고 관리
- 재고 관리
- 재무 관리
- 고객 센터
- 대시 보드
